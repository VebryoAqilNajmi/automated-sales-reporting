# Automated Sales Reporting Workflow

## Overview
This project demonstrates an automated sales reporting workflow using Python, pandas, and Selenium.

The system processes raw sales data, generates automated business summaries, exports Excel reports, and demonstrates browser automation workflows.

---

## Features
- Automated CSV data processing
- Sales summary generation
- Top category & region analysis
- Excel report generation
- Selenium browser automation

---

## Workflow

```text
CSV Dataset
↓
Python + pandas
↓
Generate Sales Summary
↓
Export Excel Report
↓
Selenium Web Automation
```

---

## Technologies Used
- Python
- pandas
- Selenium
- openpyxl
